원격 저장소 관련

------------------------------------------------------------------------------------------------------------------------------------


원격 저장소에 올리려면 git push -u 원격저장주소 브랜치이름

원격 저장소 주소 변수에 저장해서 사용하려면 git remote add origin 원격저장주소

원격 저장소 내려받기 git clone 원격저장주소
